# Virtual Doctor for Cancer Patients in VR


A virtual doctor who can discuss cancer treatment with patients. 
This project uses speech recognition using the Whisper model and [Sentis](https://unity.com/products/sentis), LLM inference using [LLMUnity](https://github.com/undreamai/LLMUnity) , spoken speech to text, animated avatar sequences, and supplementary image aids. 

![Cover image of system](DemoImage.png)



## Requirements:
- Unity 2022.3.62f1 recommended (experimental packages being used)
- Microphone permissions required
- [Git LFS](https://git-lfs.com/)
- Meta Quest 2 or 3
- OpenAI key

## Running the project
- Use microphone button or button A on controller to converse with avatar
- Application will split up message and respond as promptly as possible
- Avatar will be animated in response
